# UI Reference

## Direction

Phone-first remote companion, not server-admin chrome.

## Real references used

- ChatGPT mobile for message cadence and sticky compose ergonomics
- Tailscale admin for compact status surfaces
- Linear mobile for dense but calm list cards
- coder/code-server for workspace and session hierarchy

## Visual choices in this implementation

- dark atmospheric backdrop with cold cyan/indigo status accents
- rounded glass panels for command-center feel
- strong bottom nav for one-thumb operation
- combined chat + timeline view so approvals and diffs stay close to the conversation
