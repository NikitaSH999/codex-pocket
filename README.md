# 🚀 codex-pocket

> _Когда лежишь на кроватке, но хочешь чтобы AI работал за тебя_

Мобильное управление `codex app-server` из любого браузера.  
Открыл с телефона — и ты уже контролируешь своего агента, пока пьёшь чай.

Это не обёртка над чатом.  
Это **пультик для твоего локального Codex**, прямо в кармане.

## 🏗️ Как это работает

```
┌─────────────────────────────┐
│  📱 Телефон / Браузер       │
└──────────────┬──────────────┘
               │ WebSocket + HTTP
┌──────────────▼──────────────┐
│  codex-pocket server        │
│  (Fastify + JSON-RPC мост)  │
└──────────────┬──────────────┘
               │ stdio JSON-RPC 2.0
┌──────────────▼──────────────┐
│  codex app-server           │
│  (настоящий AI-агент)       │
└─────────────────────────────┘
```

Это не мок. Это **реальный мост** между браузером и `codex app-server`.

## ⚡ Quick start

Самый быстрый путь:

```bat
codex-pocket.bat
```

Или для красивых людей:

```bash
npm install
npm run dev
```

Прод:

```bash
npm run build && npm start
```

Для доступа по статическому IP:

```bat
codex-pocket-remote.bat
```

## 🎯 Первый запуск

1. Открыл `http://localhost:4318` (или IP из LAN)
2. Задал PIN (один раз)
3. Зашёл с телефона
4. Выбрал воркспейс
5. Лёг обратно на кроватку и работаешь 🛏️

## ⌨️ Slash-команды

Набери `/` в composer:

| Команда | Что делает |
|---------|-----------|
| `/plan` | Включить Plan mode |
| `/default` | Обычный режим |
| `/fast` | Скорость: быстро |
| `/deep` | Скорость: глубоко |
| `/fork` | Форкнуть сессию |
| `/mcp` | Статус MCP серверов |
| `/settings` | Настройки |

`Ctrl+Enter` — отправить, `Esc` — закрыть меню.

---

## 🔌 LAN и Remote доступ

**Private mode** (по умолчанию):
- localhost + LAN + приватные подсети

**Public mode** (`codex-pocket-remote.bat`):
- Открывает Windows Firewall
- Пробует UPnP маппинг
- Доступ по статическому IP

> Если провайдер за CGNAT — это проблема провайдера, а не наша 🤷

Под капотом:
